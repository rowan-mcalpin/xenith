MotorEx

open class MotorEx(val deviceName: String, val type: MotorEx.Type = Type.GOBILDA_YELLOWJACKET, val gearRatio: Double = 19.2, val resetEncoderOnInit: () -> Boolean = { true })

This class is the Xenith extension of the DcMotorEx class. It allows more customizable control of a motor, using a specific MotorController, and avoids having to manually run the hardwareMap.get() function.

Parameters

deviceName

the name of the motor, specified by the configuration in the driver station app

type

the type of internal motor used

gearRatio

the total gear ratio of the motor (internal gearing + any external gearing)
